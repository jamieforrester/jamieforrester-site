Institutional Topology // Written Diagnostic Artifacts
01 // Overview structural diagnosis

Upstream diagnosis for systems that fail under scale.

I produce written Diagnostic Artifacts that make a system’s real structure visible: what it does, what must not break, and where failure concentrates.

Most chronic complexity is not a “people problem.” It is topology: misnamed primitives, unclear invariants, and enforcement living in the wrong place.

Output
A written Artifact: primitives, invariants, enforcement points, failure modes, and a build map.
Use when
Incidents repeat, process sprawl grows, exceptions metastasize, “alignment” loops substitute for authority.
Not included
Implementation execution, delivery management, org-change programs, coaching.
