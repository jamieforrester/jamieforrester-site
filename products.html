The 60-Second Structural Scan

Available now Free

Four questions that surface the structural reality of a system before analysis begins. Purpose → Primitive → Rules → Failure. The entry point into the diagnostic method. Answer all four honestly and you will know whether the failure is structural, and where to look first. Difficulty answering any of the four questions is itself a diagnostic signal.

Run it before sending the Five-Line Brief. Run it before commissioning a Diagnostic Artifact. Run it whenever a system is producing recurring failure and the cause is unclear.
